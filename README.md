# RESPIRE
Music referencing app with an audio waveform analyzer and Spotify SDK integration

## Getting Started

Installing
npm install -g webpack webpack-cli
npm install

## Running

```
npm run build
npm start
```

Once started, server will be running at `http://localhost:8080/`
